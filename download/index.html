下载

OpenRA是一款跨平台游戏，适用于多种操作系统。 目前最新且稳定的发布版是：release-20210321

我应该选择哪一个版本？

OpenRA提供两个可选版本，分别是测试版（Playtest）和发布版（Release），你可以根据自己的需求自行选择。

  • 发布版适合那些追求稳定并且有联机需求的玩家；
  • 测试版则包含了一些可能不太稳定的更新或者改动，如果你想要帮助测试或者改进OpenRA，可以考虑游玩测试版。

游戏素材与权属说明

OpenRA引擎是基于GPL3协议的免费开源软件，而OpenRA的模组必须依赖原始游戏的素材文件才能正常运行。上述素材文件的使用均遵循EA发布的命令与征服系列模组说明书（C&C Franchise Modding Guidelines）。请注意，这些文件的权属并不由OpenRA方面持有，并且你将被要求从合法的渠道下载或取得这些文件。

EA对OpenRA不负有任何责任。

选择你的操作系统

OpenRA可在任何支持DirectX11以及OPenGL 3+的Window 7以及后续操作系统上运行

下载发布版 release-20210321 (26.44MB) 下载测试版 No Playtest Available

32位安装包和可移植版本可以从release-20210321下载。

从WinGet获得稳定发布版：

> winget install openra

从Chocolatey获得软件包：

> choco install openra

请注意，OpenRA官方不负责该社群平台的维护工作；在新版本发布后，此类下游平台可能会出现一些发布延迟

OpenRA适合在macOS 10.13及后续版本的操作系统上运行

下载发布版 release-20210321 (60.79MB) 下载测试版 No Playtest Available

支持 OS X 10.9 – 10.12 的兼容性版本可以从 release-20210321 下载。

从Homebrew Cask获得稳定发布版：

$ brew install --cask openra

请注意，OpenRA官方不负责此类社群平台的维护工作；在新版本发布后，此类下游平台可能会出现一些发布延迟

官方AppImages

OpenRA可以像其他应用程序一样在绝大部分当代64位Linux系统上运行

你必须单独下载每一个模组，并使之可被顺利执行，随后双击即可启动游戏；你也可以安装可选的appimaged程序，从而将OpenRA与你的系统或浏览器集成在一起

社区维护包

OpenRA也可以使用Linux发行版或其他第三方提供的软件包进行安装。这些软件包将提供与您的系统更好的集成，但可能并不总是提供最新版本。有关任何打包问题，请联系下游存储库维护人员。

请注意，OpenRA官方不负责此类社群平台的维护工作；在新版本发布后，此类下游平台可能会出现一些发布延迟

Flatpak

从Flathub获得稳定发布版及可玩性测试版：

$ flatpak install --from https://flathub.org/repo/appstream/net.openra.OpenRA.flatpakref $ flatpak install --from https://flathub.org/beta-repo/appstream/net.openra.OpenRA.flatpakref

请注意，Flatpak沙盒会干扰官方和社区mod之间的mod切换功能。我们建议希望使用此功能的玩家运行AppImage版本。

Snap

从Snap Store获得稳定发布版：

$ snap install openra

Arch Linux

从官方Arch 仓库获得稳定发布版：

$ pacman -S openra

openSUSE Linux
从openSUSE Build Service repository获得稳定发布版及可玩性测试版：

$ zypper install openra

Solus

从官方Solus存储库获得稳定发布版：

$ eopkg it openra

Ubuntu
从XtraDeb获得稳定发布版：

$ apt install openra

Void Linux

从官方Void Linux 仓库获得稳定发布版：

$ xbps-install openra

玩家可以在FreeBSD Ports Collection获取最新的稳定版资源

# cd /usr/ports/games/openra # make install

请注意，OpenRA官方不负责此类社群平台的维护工作；在新版本发布后，此类下游平台可能会出现一些发布延迟
